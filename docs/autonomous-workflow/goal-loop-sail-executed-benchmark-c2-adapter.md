## Intended Outcome

The repository exposes two honest executable capability surfaces:

1. A new SAIL structural benchmark version where registered candidate methods
   receive only frozen public case payloads, emit strictly validated
   predictions under deterministic budgets, and are scored by an
   evaluator-owned sealed truth and aggregation path. Oracle and full-batch
   controls remain evaluator-owned.
2. Exactly one new trusted C2 adapter that takes a result-free,
   content-addressed request, verifies the retained action tensor, owns its
   preregistered simulator mutations and executions, derives raw artifacts,
   invokes an independent CPU/fp32 consequence evaluator, and issues a
   receipt consumable by the live operator. If those prerequisites cannot be
   verified safely, the same interface terminates with a sealed prerequisite
   abstention and no fabricated result or posterior movement.

Neither surface grants training, simulator promotion, physical capture,
gateway, robot motion, transfer, or task authority.
