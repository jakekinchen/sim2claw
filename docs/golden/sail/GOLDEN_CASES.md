# SAIL Golden Cases

The machine-readable registry is
`tests/fixtures/sail/golden_cases_v1.json`. The test node recorded for each case
is the required implementation endpoint; P1-01 activates the contract and
authority subset, while later milestones activate structural and retained-data
cases without changing their expected verdicts.

| --- | --- |
| GOLD-00 | Missing or changed source fails before fitting. |
| GOLD-01 | Any action shape, dtype, order, or byte change fails invariance. |
| GOLD-02 | Valid evidence round-trips with stable digest and masks. |
| GOLD-03 | Missing camera, object, or contact channels are `not_evaluable`. |
| GOLD-04 | A phase shift is detected even when trajectory minima match. |
| GOLD-05 | Compensation debt triggers structural surprise. |
